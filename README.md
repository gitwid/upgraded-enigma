# upgraded-enigma
Digital Mindfulness

Also in this repository:

- [Morningstar](morningstar/) — a local-first, provenance-preserving
  observational instrument (Python reference implementation; see
  `morningstar/README.md`).
- [iOS app](ios/) — the native Morningstar app: `MorningstarKit` (a Swift port
  of the domain core, verified against the reference implementation's RFC 8785
  golden vectors) plus a SwiftUI shell that wears this site's navigational
  principle as its capture flow (Breath → Mind → Eye → Play). See
  `ios/DESIGN.md`.

The static Digital Mindfulness site (`index.html`, `style.css`) remains — it is
now the design source for the iOS app's experience.
